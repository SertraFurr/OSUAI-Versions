I don't know why, but here's a way to get all existing version of https://osu.ai (Do not support any of the guys behind.)

As of 02/21/2026


To download any version simply update the download url with the version.

https://osu.ai/api/download-desktop-client?url=https://cdn-releases.osu.ai/OsuAI_VERSION_x64_en-US.msi

Exemple: https://osu.ai/api/download-desktop-client?url=https://cdn-releases.osu.ai/OsuAI_1.7.8_x64_en-US.msi
